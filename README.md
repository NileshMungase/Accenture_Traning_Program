# Accenture Training Program

This repository contains exercises and small projects completed during the Accenture training program.

## Repository structure

- Calculator/
  - A calculator project (HTML/CSS/JS). Open the files in this folder in a browser to try it.
- portfolio/
  - Personal portfolio site files. Open the contained HTML files in a browser.
- pratice/
  - Practice exercises and snippets (typo preserved from original folder name).
- index.html
  - Landing page for the repository. You can open this file in a browser to view the main page.
- script.js
  - JavaScript used by some pages in the repo (including the landing page).

## How to run

To view the projects locally, you can simply open the HTML files in your browser. For a simple local server, run:

```
# Python 3
python -m http.server 8000
```

Then open http://localhost:8000 and navigate to the relevant folder (for example `Calculator/` or `portfolio/`).
